#Coda 2 Parser Mode

##Requirements

	Coda 2 (http://panic.com/coda/)
	Parser 3.4.2 (http://www.parser.ru/en/, http://parser.ru)

##Installation

	1. Close Coda 2
	2. Copy Parser.mode to ~/Library/Application Support/Coda 2/Modes
	3. Open Coda 2
	4. Switch 'Syntax mode' to 'Parser'
